### Données

Un déclencheur contient les champs suivants :

#### Général

- Nom du déclencheur
- Icône : icône du déclencheur

#### Périodicité (liste)

- Identifiant : identifiant de l'association avec la crontab
- Ordre : ordre d'affichage
- Nom : nom de la crontab
